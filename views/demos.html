simuL8r Demos


What is it? - simuL8r is self explanatory once you see the demos, its basically a physics engine that lets users create fun and interactive scenes,
eventually when the app is complete it be should be possible to use it to make simple games.

This page - This project is still at an early stage, and many of the features have yet to be built but the purpose of this page
is to act as a demonstration for the kind of things that can be already made with this app.

Instructions - simply click on any of the thumbnails below to access a demo, and follow the instructions when you get there (its very simple, just click the play button)

Create your own scenes - select the "make a scene!" tab on the left hand navigation and when you get to the canvas hover over the icons to learn how they work.

To view the code or to read more about this project please visit GitHub
